Instructions

You can find the source code for this system on GitHub.

To use it on your own website, add the following to the header of each page that you want to use it on:

<link rel="stylesheet" href="https://design.example.com/{{ version path }}/index.css" crossorigin="" />

Then, add the following to the end of each page that you want to use it on, just before you close your <body> tag:

<script src="https://design.example.com/{{ version path }}/bundle.js"></script>
